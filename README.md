# Java, Spring & Hibernate Learning Repository

## Overview
This repository contains a collection of Java EE, Spring Framework, Spring Boot, Hibernate, Microservices, and Web Application experiments.  
Each folder represents a **separate hands-on example or experiment** covering core to advanced backend technologies.

The projects are mainly intended for **practice, learning, and academic/lab use**.

---

## Technology Stack
- Java (Core & Advanced)
- Servlet & JSP
- JSTL & Custom Tags
- Hibernate ORM
- Spring Core, Spring MVC, Spring Boot
- Spring Security, Spring AOP
- Spring Cloud & Microservices
- REST APIs
- JDBC
- H2 & MySQL
- Apache Tomcat 10.1
- Maven
